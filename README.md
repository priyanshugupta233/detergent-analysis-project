# 🧼 Detergent Analysis Project

## 📊 Overview
This project analyzes the efficiency of different detergents based on cleaning score, time, and amount used.

## 🔍 Steps Performed
- Data Collection
- Data Cleaning
- Data Analysis
- Data Visualization
- Basic Machine Learning

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## 📈 Results
- Ariel performed best overall
- Ink stains are hardest to clean
- Proper time and amount improve cleaning
